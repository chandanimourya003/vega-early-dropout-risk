# vega-early-dropout-risk
# AI-Based Early Dropout Risk Prediction & Intervention System

## Problem Statement
Develop an AI-powered system to predict students at risk of early dropout and recommend appropriate interventions.

## Solution Overview
This system uses Machine Learning to analyze multiple student performance indicators such as attendance, academic scores, and engagement metrics to predict dropout probability.

The system classifies students into:
- Low Risk
- Medium Risk
- High Risk

Based on risk level, targeted intervention strategies are recommended.

## AI Approach
- Model Used: Logistic Regression / Random Forest
- Data Split: 70% Training, 30% Testing
- Output: Dropout Probability Score
- Risk Classification based on probability thresholds

AI is core to this system because dropout prediction is based on learned patterns from multi-factor data rather than hardcoded rules.

## Key Features
- Student Data Input Form
- Risk Prediction Engine
- Risk Categorization
- Automated Intervention Suggestions
- Admin Dashboard
